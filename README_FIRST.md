# Сначала прочитай это

Это упрощённая версия без папок. В репозитории GitHub должно быть ровно 5 основных файлов в корне:

- `Dockerfile`
- `main.py`
- `requirements.txt`
- `render.yaml`
- `.gitignore`

## Переменные Render

Обязательные:

- `ADMIN_PASSWORD` — пароль входа на сайт
- `TWOGIS_API_KEY` — ключ 2ГИС
- `DATA_LICENSE_ACKNOWLEDGED` — `true` только если договор/тариф 2ГИС разрешает сохранять данные

Необязательные:

- `TELEGRAM_BOT_TOKEN` — токен бота от BotFather
- `TELEGRAM_ADMIN_CHAT_ID` — твой числовой chat ID; бот будет присылать тебе уведомления о завершении задач

`DATABASE_URL` и `SESSION_SECRET` Render создаст сам через `render.yaml`.

## Что умеет Lite

- веб-панель с паролем;
- поиск компаний через 2ГИС;
- режим «нет сайта, но есть контакты»;
- режим технического аудита существующих сайтов;
- таблица лидов;
- выгрузка Excel;
- уведомления владельцу в Telegram.

Бот Telegram не может первым писать произвольным компаниям: Bot API разрешает отправку только пользователям/чатам, которые уже взаимодействовали с ботом.
