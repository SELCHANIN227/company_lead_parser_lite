# Company Lead Parser — Yandex v2

Обновление исправляет главный дефект предыдущей версии: задача больше не помечается как успешная, если Яндекс не отдал ни одной распознаваемой карточки.

## Что изменено

- данные извлекаются не только по одному CSS-селектору;
- перехватываются JSON-ответы, которые загружает сама страница Яндекс Карт;
- есть несколько DOM-селекторов и сбор ссылок на карточки;
- карточки обогащаются при отдельном открытии;
- при нуле результатов задача получает статус `ошибка`;
- сохраняются скриншот и HTML проблемной страницы;
- на странице задачи появляются кнопки для открытия диагностики;
- сохраняется подробная статистика по каждой категории.

## Обновление существующего Render-сервиса

В GitHub-репозитории `company_lead_parser_lite` замените файл `main.py` файлом из этого архива. Остальные настройки Render и ключи менять не нужно. Render автоматически выполнит новый deploy.

Для первого теста:

- источник: Яндекс Карты;
- режим: Все карточки;
- город: Воронеж;
- категория: стоматология;
- максимум: 5.

Если Яндекс отдаст заглушку, CAPTCHA или изменённую страницу, задача будет помечена ошибкой и покажет скриншот/HTML вместо ложного `готово, 0`.
