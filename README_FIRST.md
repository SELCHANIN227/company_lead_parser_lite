# Версия с Яндекс Картами без API

Главное изменение: в форме появился источник **«Яндекс Карты — без API»**.
Парсер запускает Chromium через Playwright, прокручивает видимый список, открывает карточки и собирает доступные название, адрес, телефон, email, сайт, рейтинг и ссылку на карточку.

## Важно

- CAPTCHA не обходится. Если она появится, задача остановится с понятной ошибкой.
- Нет маскировки браузера, прокси-ротации и обхода ограничений.
- Интерфейс Яндекс Карт может измениться; тогда селекторы понадобится обновить.
- На бесплатном Render Chromium может работать медленно. Первый тест: одна категория, максимум 5–10.

## Как обновить существующий GitHub-репозиторий

Замени в корне репозитория три файла:

- `main.py`
- `requirements.txt`
- `Dockerfile`

`render.yaml` можно оставить прежним. Новых API-ключей для Яндекса не требуется.

После commit Render должен автоматически начать новый deploy. Если не начал: Render → сервис → Manual Deploy → Deploy latest commit.

## Режимы

- **Все карточки — для проверки**: сохраняет всё, что удалось прочитать.
- **Без сайта, но с контактом**: только карточки без сайта и с телефоном/email/мессенджером.
- **Есть сайт — технический аудит**: только карточки с URL сайта.

Каждая задача теперь показывает диагностику: сколько карточек найдено, сколько имеют сайт, контакты и сколько прошли фильтр.
